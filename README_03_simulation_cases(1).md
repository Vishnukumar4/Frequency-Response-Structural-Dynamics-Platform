# README 03: Simulation Cases

This document describes the simulation cases used in the **Frequency Response Structural Dynamics Platform**.

Each case was designed to build understanding step by step, starting from a simple single-degree-of-freedom system and moving toward FEA-inspired structural and vibro-acoustic simulations.

---

## Case 1: Single Degree-of-Freedom Oscillator

### Objective

Validate the frequency response solver using a simple mass-spring-damper system.

### Model

```text
m*x'' + c*x' + k*x = F(t)
```

### Inputs

| Parameter | Description |
|---|---|
| `m` | Mass |
| `c` | Damping coefficient |
| `k` | Stiffness |
| `F` | Harmonic excitation force |

### Output

- Natural frequency
- FRF curve
- Resonance peak
- Damping effect

### Purpose

This case verifies that the numerical frequency response matches the analytical solution:

```text
H(omega) = 1 / (k - m omega^2 + j c omega)
```

## Case 2: Two Degree-of-Freedom Structural System

### Objective

Study coupled vibration behavior between two masses.

### Model

A two-mass, spring-damper system was used to show how multiple natural frequencies appear in the FRF.

### Output

- Two natural frequencies
- Two mode shapes
- Frequency response with two resonance peaks
- Modal interaction behavior

### Learning Outcome

This case helped connect eigenvalue analysis to physical vibration behavior.

## Case 3: Multi-DOF FEA-Inspired System

### Objective

Use mass and stiffness matrices similar to those generated by finite element analysis.

### Inputs

```text
mass_matrix.csv
stiffness_matrix.csv
damping_matrix.csv
excitation_config.json
```

### Output

- Natural frequency table
- Mode shape vectors
- FRF at selected response DOFs
- Resonance summary

### Learning Outcome

This case demonstrated how FEA matrices can be used directly for dynamic response simulation.

- Distributed mass
- Boundary constraints
- Modal response

### Output

- Higher bending modes

- Automotive body panels

## Case 5: Vibro-Acoustic Response Interpretation

### Objective

Connect structural vibration response to sound radiation and acoustic transmission.

### Concepts Used

- Surface vibration
- Velocity response
- Resonance-driven acoustic radiation
- Transmission loss interpretation
- Structural-acoustic coupling

### Output

- Surface velocity vs frequency
- Resonance regions with likely acoustic radiation
- Transmission loss trend interpretation

### Learning Outcome

This case helped show that large structural vibration response can lead to stronger acoustic radiation.

---

## Case 6: SEA Concept Study

### Objective

Explore how high-frequency vibro-acoustic systems can be interpreted using Statistical Energy Analysis concepts.

### Concepts Used

- Modal density
- Subsystem energy
- Damping loss factor
- Coupling loss factor
- Power flow

### Output

- Energy-flow interpretation
- High-frequency response trend
- Conceptual comparison between deterministic FEA and SEA

### Learning Outcome

This case showed why deterministic modal analysis is useful at low frequency, while SEA is useful at higher frequency where many modes contribute.
